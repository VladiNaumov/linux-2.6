Приложение В, содержащее исходный код Minix, находится в файлах LISTING.txt и LISTINGrus.txt, LISTING.PDF.

СОДЕРЖИМОЕ CD:

Этот CD содержит:

- README.TXT Этот файл
- SETUP.PS: руководство по настройке в PostScript
- SETUP.PDF: руководство по настройке в PDF (требуется Adobe Acrobat Reader)
- PRESZ134.ZIP Утилита Windows для изменения размера разделов FAT
- BOOTFLOP.IMG Если вы не можете загрузиться с CD-ROM, вставьте CD-ROM в любом случае, затем скопируйте этот образ дискеты в формате raw на дискету с помощью RawWrite (спросите Google) и загрузитесь с дискеты
- LISTING.TXT Приложение B как текст ASCII
- LISTING.PS Приложение B в PostScript
- LISTING.PDF Приложение B в PDF
- INDEX.TXT Индекс файлов
- BOOKSRC.TGZ сжатый с помощью gzip файл UNIX tar с файлами, перечисленными в книге плюс измененные Makefiles и дополнительные файлы для компиляции подмножества книги Minix.

Также для установки MINIX 3 используется множество невидимых файлов.
